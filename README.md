## What is RudyGuard?

**RudyGuard** is a blazing-fast, modular, and visually advanced network monitoring/security toolkit for Linux and Parrot OS.  
Built from scratch by **Rudy Cooper (@FrozenBarbie01)**, this tool is for hackers, sysadmins, and anyone who wants deep insight into their network traffic‚Äîwith style, colors, emojis, geo info, and more.

---

## Features

- üî• **Live Network Monitoring** (TCP/UDP, LAN/Public)
- üåç **True Geolocation** (Country, Region, ISP, Domain)
- üè† **LAN Detection** with special labels
- üö® **Real-Time Alerts** (Terminal, Desktop, Sound)
- üö¶ **Traceroute** (Colorful, emoji-powered hops)
- üìä **Advanced Logging** (JSON logs for all activity)
- üß† **Heuristics/AI Ready** (Suspicious IPs, anomaly scoring, future ML support)
- üé® **Modern UI** (ASCII logo, colors, emojis, banners)
- üí° **Easy to Extend** (Modular code, plug in your own logic)
- üõ°Ô∏è **Security Focused** (Ready for Parrot OS, root-friendly)
- üïí **Timestamped Events**

## Installation

**Clone the repository:**
```bash
git clone https://github.com/FrozenBarbie01/RudyGuard.git
cd RudyGuard
```

**Install dependencies:**
```bash
pip install -r requirements.txt
```

**Run RudyGuard:**
```bash
python3 RudyGuard.py
```

---

## Usage

- **Requires Python 3.8+**
- **Best run as root** for full network visibility
- **Works on Parrot OS, Kali Linux, Ubuntu, Debian**
